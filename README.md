## Features

Excaliban kanban board support:

- 🗃&nbsp;Multiple Boards.
- 📚&nbsp;Task Colors.
- 📢&nbsp;Task Priorities.
- 📆&nbsp;Task Due Dates.
- 💾&nbsp;Exports & Imports.
- 💯&nbsp;Free & Open-Source.
- ✍️&nbsp;Hand-drawn like style.
- 🌓&nbsp;Dark mode.
- 👅&nbsp;Localization.
